Defense in depth

Four reasons the installer is safe

Open source, MIT licensed

Every line of code that goes into the installer is on GitHub. Anyone can read it, fork it, audit it, or build their own bytewise‑identical installer from source.

SHA‑256 published

Each release lists a SHA‑256 hash for every artifact, both in latest.json and on this page. If a CDN ever served a tampered file, the hash would change.

VirusTotal cloud scan

The links above query the live report for each installer's SHA‑256 hash on virustotal.com — so the verdict is independently checkable by 70+ antivirus engines, not just our word.

No telemetry, no account

The app never phones home with analytics, crash reports, or usage data. Your Claude API key, your conversations, and your project files all stay on your machine — and the encrypted vault uses Windows DPAPI.

Full disclosure

Every outbound network call the app makes

"No telemetry" means we don't track you — not that the app never opens a socket. Here is every third‑party endpoint Iris Code talks to, and exactly when, so you can verify it with a packet sniffer or your firewall logs.

api.anthropic.com

Every chat turn. The Claude Code CLI sends your prompt + context to Anthropic and streams the model response back. This is the core function of the app — without it there is no agent. Uses your own API key or Pro/Max subscription session.

iris-code.dev/latest.json

Once per launch (best‑effort, never blocks startup). Returns the latest published version + download URL so the update banner can light up when a newer release is out. Disable by blocking the host in your firewall — the app just won't show the banner.

www.google.com/s2/favicons

Per visible web‑result card. When the orchestrator returns a web search citation, the renderer shows a rich card with a 20×20 favicon fetched from Google's public s2/favicons endpoint, scoped by the citation's hostname. The request sends the cited site's domain to Google (no user identifier, no API key). Block the host or run offline and the cards still work — they just render with an empty icon box. Source: app/css/web-cards.css + app/js/lib/markdown.js.

Optional integrations (off unless you enable them)

Telegram bot API (only when you pair a bot), the MCP marketplace catalog (only when MCP is enabled), and any user‑configured remote access listener you choose to expose. None of these run by default.

"But Windows SmartScreen still warns me…"

SmartScreen warns about any installer it hasn't seen many users run before — it's a reputation system, not a malware scanner. The v0.4.0 build isn't code‑signed yet (a Windows EV certificate costs $300+/year and we're holding off until v0.5), so it starts at zero reputation regardless of what's inside.

That's exactly why this page exists. The VirusTotal scans above are the real, content‑based verification — SmartScreen is just reputation.
